機能

6つの解析レイヤー

写真の真正性を多角的に検証するため、6つの独立した解析を組み合わせています。

📋

EXIFメタデータ解析

カメラ機種・GPS座標・撮影日時・ソフトウェア情報を完全記録。改ざん・削除の痕跡を検出します。

🔍

ELA 誤差水準解析

JPEG 再圧縮時の誤差パターンを解析し、画像編集ソフトによる加工箇所を高精度で可視化します。

🤖

AI生成画像検出

HuggingFace の最先端モデルで AI 生成写真を識別。未使用時はローカルの FFT 周波数解析にフォールバック。

⏱

RFC 3161 タイムスタンプ

国際標準（RFC 3161）に準拠した認定タイムスタンプ。FreeTSA・DigiCert・Sectigo の3局から選択可能。

⛓

ブロックチェーン記録

写真ハッシュと真正性スコアを Polygon に永久記録。改ざん不可能な第三者証明を実現します。

📄

PDF証明書発行

全解析結果・タイムスタンプ・ブロックチェーンリンクを含む A4 証明書を即時生成。法的書面として提出可能。

仕組み

3ステップで完了

01

写真を送信

REST API の POST /verify エンドポイントに、写真ファイルと API キーを送信するだけ。

bash
curl -X POST https://api.imprint.jp/verify \
  -H "X-API-Key: imp_xxxxxxxx" \
  -F "file=@photo.jpg"
02

多層解析を実行

EXIF・ELA・AI 検出・SHA-256 を並列処理。タイムスタンプとブロックチェーン記録は任意で追加できます。

損害保険

保険詐欺を写真解析で防ぐ

事故・水災・火災時に提出された写真を即座に解析。加工の有無をスコアで可視化し、 損害査定の精度と信頼性を向上させます。

  • 査定フローへの API 統合
  • 加工検出でアラート発報

不動産

物件写真の広告コンプライアンス

掲載写真の ELA スコアと EXIF データを自動チェック。過度な修正・AI 生成物件写真を検出し、 消費者への正確な情報提供を担保します。

  • 物件登録時に自動検証
  • 撮影日時・場所をブロックチェーン記録
  • コンプライアンス証明書を自動発行

証拠写真の法的証明力を強化

  • 証拠写真に時刻証明を付与
  • Polygonscan でリアルタイム確認
  • A4 PDF 証明書を裁判所に提出
