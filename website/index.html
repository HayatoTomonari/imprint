選ばれる理由

なぜ Imprint が選ばれるのか

写真の真正性証明に特化した専門プラットフォームだからこそ、業界固有の要件に応えられます。

01

6層の多角的解析で、見落とさない

EXIF・ELA・AI検出・SHA-256・タイムスタンプ・ブロックチェーンの6レイヤーを組み合わせることで、単一手法では気づけない加工を検出。0〜100点のスコアで客観的に評価します。

機能

6つの解析レイヤー

写真の真正性を多角的に検証するため、6つの独立した解析を組み合わせています。

EXIFメタデータ解析

カメラ機種・GPS座標・撮影日時・ソフトウェア情報を完全記録。改ざん・削除の痕跡を検出します。

ELA 誤差水準解析

JPEG 再圧縮時の誤差パターンを解析し、画像編集ソフトによる加工箇所を高精度で可視化します。

AI生成画像検出

HuggingFace の最先端モデルで AI 生成写真を識別。未使用時はローカルの FFT 周波数解析にフォールバック。

RFC 3161 タイムスタンプ

国際標準（RFC 3161）に準拠した認定タイムスタンプ。FreeTSA・DigiCert・Sectigo の3局から選択可能。

ブロックチェーン記録

写真ハッシュと真正性スコアを Polygon に永久記録。改ざん不可能な第三者証明を実現します。

PDF証明書発行

全解析結果・タイムスタンプ・ブロックチェーンリンクを含む A4 証明書を即時生成。法的書面として提出可能。

FAQ

Starter（無料）プランでは、月50回まで EXIF解析・ELA解析・AI生成検出・SHA-256ハッシュ・PDF証明書の全機能が利用できます。RFC 3161 タイムスタンプとブロックチェーン記録も含まれます。

シンプルな REST API（multipart/form-data）のため、一般的な開発者であれば数時間で統合可能です。API ドキュメントと curl サンプルを提供しています。

写真の実データはサーバーに保存されません。保存されるのは SHA-256 ハッシュ値・EXIF要約・タイムスタンプトークン・ブロックチェーントランザクション情報のみです。個人情報保護の観点から、元画像は処理後に即時破棄します。

RFC 3161 は国際標準（IETF）の電子タイムスタンプ規格で、認定 TSA（タイムスタンプ局）の電子署名が付与されます。日本の電子帳簿保存法にも対応した形式です。ただし法的有効性は状況により異なるため、重要な案件では法律の専門家へのご相談をお勧めします。

HuggingFace API を使用する場合、最先端の AI 生成検出モデルを適用します。ただし AI 検出技術は生成モデルの進化に伴い精度が変化します。Imprint のスコアはあくまで参考指標であり、他の解析レイヤー（EXIF・ELA）と組み合わせて判断されることをお勧めします。

Enterprise プランではオンプレミス対応・プライベートクラウドへのデプロイについてもご相談を承ります。セキュリティポリシーや規制要件に応じたカスタム構成が可能です。お問い合わせフォームよりご連絡ください。
